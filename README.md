# MVP BackEnd
 
> Criação de um sistema para facilitar a busca pela biodivercidade dos parques de Teresopolis.

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** Node.js (Javascript) / React.js
* **Banco de Dados:** SQLite

## 🚀 Como Executar o Projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
* [Git](https://git-scm.com)
* [Node.js](https://nodejs.org/en/) (versão 18+ ou superior)

### Passos para Instalação

1. Clone este repositório:
   ```bash
   git clone [https://github.com/Gabriel-Garcia01/Back-EndMVP/edit/master.git](https://github.com/Gabriel-Garcia01/Back-EndMVP/edit/master.git)

## Acessar pasta do projeto

cd Back-EndMVP

## Instalar as dependencias

npm install  # ou pip install -r requirements.txt

## Iniciar o servidor

npm start # ou node serve.js
